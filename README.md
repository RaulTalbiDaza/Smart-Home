# Smart-Home

## 📄 Descripción del proyecto 
Este proyecto consiste en el desarrollo de una landing page que simula un dashboard de control para un hogar inteligente (Smart Home).

La interfaz representa un panel visual con distintos widgets que muestran información relevante del sistema, como:
- Temperatura  
- Control de luces  
- Consumo energético  
- Cámaras de seguridad  

El objetivo es recrear la apariencia y organización de una aplicación real de domótica, ofreciendo una experiencia visual clara, moderna y estructurada.

El proyecto está desarrollado únicamente con HTML y CSS, sin uso de JavaScript, y presenta las siguientes características:
- Diseño responsive adaptado a diferentes dispositivos  
- Modo día/noche implementado mediante variables CSS  
- Estructura basada en componentes tipo dashboard  

Este proyecto sirve para practicar conceptos clave de desarrollo web, como la maquetación, el uso de estilos avanzados en CSS y la organización de interfaces visuales.

## 🚀 Cómo ejecutar el proyecto (Deploy)  
Para poder visualizar el proyecto en tu ordenador, sigue estos pasos:

1. Clonar o descargar el repositorio  
Puedes clonar el repositorio con el siguiente comando:

git clone https://github.com/RaulTalbiDaza/Smart-Home.git

O descargarlo directamente en formato ZIP desde GitHub.

2. Acceder a la carpeta del proyecto  
Una vez descargado, abre la carpeta donde se encuentran los archivos.

3. Ejecutar el proyecto  
Abre el archivo `index.html` en tu navegador web.

Puedes hacerlo de varias formas:
- Doble clic sobre el archivo  
- Arrastrándolo al navegador  
- Click derecho → “Abrir con” → Navegador  

No es necesario instalar dependencias ni configurar servidores.

📌 Ejemplo de uso  
Al abrir el proyecto, se mostrará un dashboard de Smart Home donde podrás visualizar:

- Estado de la temperatura del hogar  
- Control visual de luces  
- Indicadores de consumo energético  
- Vista simulada de cámaras de seguridad  

Todo ello dentro de una interfaz responsive con opción de modo día/noche.
