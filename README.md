# 🏠 Smart Home Dashboard

## 📄 Description

Landing page simulating a smart home control dashboard, built with HTML and CSS.
Displays key home system information through a clean, modern interface.

**Widgets included:**
- Temperature monitor
- Light controls
- Energy consumption indicators
- Security camera view

**Key features:**
- Fully responsive design
- Day/night mode via CSS variables
- Dashboard-style component layout
